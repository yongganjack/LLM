# Week07 文件说明文档

本目录实现“中文课程资料问答 Agent”。课程问题会先检索本地资料并重排相关片段，再调用模型生成结论，最后由本地代码补充引用、工具状态和运行日志。

## 核心文件

| 文件 | 用途 | 是否直接运行/修改 |
|---|---|---|
| `simple_agent.py` | Agent 主程序：意图识别、资料检索与重排、上下文组装、模型调用、四段式回答、JSONL 日志和 CLI。 | 可运行；核心实现文件。 |
| `local_knowledge.md` | 唯一的运行时课程资料来源，包含提示词模板、上下文窗口、RAG、Agent 与工具调用、使用边界。 | 按课程资料需要维护。 |
| `test_simple_agent.py` | 标准库自动测试。模型调用使用替身，不访问真实 API、Ollama 或网络。 | 可直接运行。 |
| `agent_function_spec.md` | 当前功能、工作流、边界、输出格式、日志和 CLI 参数说明。 | 阅读为主；功能变更时同步更新。 |
| `test_questions.md` | 手工验收命令、预期输出、日志查看和模型失败验证说明。 | 阅读或复制命令执行。 |

## 运行记录与版本控制

| 路径 | 用途 | 版本控制规则 |
|---|---|---|
| `runs/agent_runs.jsonl` | 每次 Agent 调用生成的一行 JSONL 运行记录。 | 由 `runs/*.jsonl` 忽略，不提交。 |
| `runs/.gitkeep` | 保留空的运行日志目录。 | 应提交。 |
| `.gitignore` | 忽略本周的运行日志文件。 | 应提交。 |
| `__pycache__/` | Python 自动生成的字节码缓存。 | 不需要提交。 |

## 常用命令

```powershell
# 自动测试：模型调用会被替身隔离
& "E:\Anaconda\envs\research\python.exe" week07\test_simple_agent.py

# 课程资料检索与模型生成
& "E:\Anaconda\envs\research\python.exe" week07\simple_agent.py --no-stream "什么是 RAG？"

# 安全模拟本地资料读取失败
& "E:\Anaconda\envs\research\python.exe" week07\simple_agent.py --no-stream --simulate-tool-failure "什么是 RAG？"
```

课程问答默认 provider 为 `cloud`，需要相应的模型服务和凭证；问候、帮助、范围外说明和模拟工具失败不调用模型。
