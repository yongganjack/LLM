# Week06 Local Knowledge For Simple Agent

本文件是 Week06 简单 Agent 的本地知识资料源。Agent 的唯一工具只读取这个文件，不访问网络，不读取其他目录，也不修改任何文件。

---

## 1. 简单 Agent 的定义

本项目中的简单 Agent 不是完整的外部 Agent 平台，也不是复杂的多工具自动规划系统。

它是一个最小可运行原型，负责完成以下固定流程：

1. 接收用户输入。
2. 判断问题是否需要额外资料。
3. 如果需要，调用一个本地工具读取资料。
4. 根据用户问题和工具结果，组织最终回答。

这个 Agent 的学习重点是理解 Agent 的基本组成方式，而不是依赖现成外部 Agent 产品代替实现。

---

## 2. Agent 的核心组成

### 输入

用户输入一段自然语言问题或任务说明，例如：

```text
请根据本地资料说明 Agent 的工作流程
```

### 判断逻辑

Agent 使用简单规则判断是否需要调用工具。

如果问题中出现以下关键词之一，Agent 应调用本地知识工具：

```text
资料
知识库
文档
根据本地
本地资料
week06
Agent
工具
流程
依据
```

如果没有出现这些关键词，Agent 可以不调用工具，直接基于聊天机器人能力回答。

### 工具

Agent 只允许调用一个工具：

```text
read_local_knowledge
```

工具作用：

- 读取当前文件 `week06/local_knowledge.md`。
- 返回文件中的文本资料。
- 作为回答依据的一部分。

工具边界：

- 不联网。
- 不调用外部 Agent 平台。
- 不写入文件。
- 不扫描整个项目。
- 不读取 `week03/`、`week04/`、`week05/` 中的文件。

### 输出

Agent 最终输出必须固定为三部分：

```text
结论：
<最终回答>

依据：
<说明回答依据，包含是否使用本地资料>

是否调用工具：
<是 / 否 / 是，但失败>
```

## 3. 完整流程示例

用户输入：

```text
请根据本地资料说明 Agent 的工作流程
```

Agent 判断：

```text
问题包含“根据本地”“资料”“Agent”“流程”，需要调用工具。
```

工具调用：

```text
read_local_knowledge -> 读取 week06/local_knowledge.md
```

最终输出：

```text
结论：
这个简单 Agent 的工作流程是：接收用户输入，判断是否需要额外资料，需要时读取本地知识文件，然后按固定格式组织回答。

依据：
依据来自 week06/local_knowledge.md，其中说明了 Agent 的定义、判断逻辑、工具边界和输出格式。

是否调用工具：
是
```

## 4. 不调用工具的示例

用户输入：

```text
什么是 Prompt Template？
```

Agent 判断：

```text
问题没有明确要求读取本地资料，也没有命中工具关键词，可以不调用工具。
```

最终输出：

```text
结论：
Prompt Template 是一种提示词模板，用于把固定的系统要求和动态输入组合成可以发送给模型的完整提示。

依据：
未调用本地工具，基于聊天机器人当前能力回答。

是否调用工具：
否
```

## 5. 工具失败处理

如果本地知识文件不存在、路径错误或读取失败，Agent 不应崩溃。

失败时固定输出：

```text
结论：
未获取到资料。以下是当前可回答部分：<基于用户问题给出的有限回答>

依据：
本地工具读取失败：<错误信息>

是否调用工具：
是，但失败
```

失败处理目标：

- 保证 CLI 可以正常结束。
- 明确告诉用户资料没有获取成功。
- 保留当前仍能回答的部分。
- 不隐藏工具失败状态。

---

## 6. 功能边界

本 Agent 可以做：

- 接收一个用户问题。
- 判断是否读取本地资料。
- 读取一个固定本地知识文件。
- 把用户问题和工具结果组织成三段式回答。
- 在工具失败时返回降级结果。

本 Agent 不做：

- 不做多工具规划。
- 不做联网搜索。
- 不调用外部 Agent 平台。
- 不引入向量数据库。
- 不自动修改项目代码。
- 不执行任意系统命令。
- 不做长期记忆。

---

## 7. 验收重点

Agent 至少应完成一次完整流程：

```text
接收任务
  -> 判断是否调用工具
  -> 调用本地知识工具
  -> 获取资料
  -> 组织固定格式回答
```

推荐验收问题：

```text
请根据本地资料说明 Agent 的工作流程
```

预期：

- 命中工具调用条件。
- 成功读取本文件。
- 输出包含 `结论`、`依据`、`是否调用工具`。
- `是否调用工具` 的值为 `是`。
